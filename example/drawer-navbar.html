Bootstrap Off-Canvas

The more common name for a drawer menu, I learned, is "off-canvas"; but, that name was taken for other packages that also did this. Even so, the current bootstrap package (3.3.2) doesn't have any off-canvas elements by default even though it is becoming a growing mobile design trend. Granted, this might change in the future, but - hey - I made this for now.

This page contains a basic example of how to use the full page. See the documentation page for more details and more examples!

If you have any issues or questions, please check out out the repo and drop an issue.
